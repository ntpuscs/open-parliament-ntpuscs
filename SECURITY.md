# 資訊安全政策

「北大三峽議事資訊網」負責呈現國立臺北大學三峽校區學生議會的重要議事資料。雖然網站展示的內容屬於公開資訊，但系統最初運作依賴具備讀寫權限的 Google 服務帳戶金鑰。保護這些憑證是本專案最優先的安全考量。

## 金鑰保護規範

1.  **絕對禁止提交憑證** ：任何人（包含學生議會人員與外部貢獻者） **絕對不可** 將包含真實金鑰的 `.env` 檔案，或是包含任何私鑰字串的 JSON 檔案 commit 至 Git 等版本控制系統中。
2.  **檢查 Git 狀態** ：在執行 `git commit` 前，請務必確認您的 `.gitignore` 檔案已正確排除 `.env` 與任何 `.json` 憑證檔。
3.  **環境變數傳遞**：生產環境的金鑰僅能透過 Cloudflare Pages 的後台環境變數或 GitHub 的 Secrets 介面進行設定，不得寫死 (Hardcode) 於專案原始碼。

## 漏洞回報機制

如果您在專案中發現了潛在的安全漏洞（特別是金鑰外洩或未經授權的資料寫入漏洞），懇請配合：

1.  請不要在 GitHub Issues 開立公開的漏洞工單。
2.  立即透過電子郵件聯絡學生議會秘書處（給秘書處：如果不知道怎麼處理，趕快找學生會資訊部或架設本專案的第26屆求救，你們的議事交接手冊應該有附聯絡方式）。
3.  **應變措施** ：一旦確認金鑰外洩，維護團隊請立即至 Google Cloud Console 撤銷該服務帳戶的憑證，並重新核發新金鑰以部署。
